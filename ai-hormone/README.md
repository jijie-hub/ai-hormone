# ai-hormone

这是一个教学演示的 React 应用：**AI辅助下的激素分级与反馈调节系统**。
界面语言为中文，适合在平板或电脑上用于公开课演示（沪教版生物·体液调节部分）。

## 运行（本地开发）
1. 解压后进入项目目录：
```bash
cd ai-hormone
```
2. 安装依赖：
```bash
npm install
```
3. 启动开发服务器：
```bash
npm start
```
浏览器会打开 `http://localhost:3000`。

## 注意
- 此项目为教学演示简化模型，旨在帮助学生理解“分级调节”和“负反馈调节”的概念。
- 如果你打算部署到公网或做更复杂的功能（如后台AI分析），需要额外添加后端与安全验证。
